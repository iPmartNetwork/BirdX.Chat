# README screenshots

Add PNG or WebP captures of the **BirdX Chat** web app here for `README.md` / `README.fa.md`.

| File | Suggested content |
|------|-------------------|
| `chat.png` | Main chat UI (channel or DM, RTL FA preferred for FA README) |
| `admin.png` | Admin dashboard overview |

After adding files, the image gallery in the root README files will render on GitHub.
